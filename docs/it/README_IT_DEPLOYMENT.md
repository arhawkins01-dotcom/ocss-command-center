
# IT Deployment Overview

- Python 3.10+
- Internal network hosting
- Access to S:\OCSS\CommandCenter

ReportLibrary: S:\OCSS\CommandCenter\ReportLibrary
Exports: S:\OCSS\CommandCenter\Exports

No database or external cloud hosting required.
