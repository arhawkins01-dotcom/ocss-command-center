## User Manual Overview

The [USER_MANUAL.md](./USER_MANUAL.md) provides comprehensive documentation for all user roles:

### Covered Roles

- **Director & Deputy Director** - Executive dashboards and leadership analytics
- **Program Officers** - Report ingestion, caseload oversight, agency-wide KPIs
- **Supervisors** - Team performance monitoring and QA review
- **Team Leads** - Caseload assignment and workflow management
- **Support Officers** - Report processing and caseload completion
- **Administrative Specialists** - Help ticket intake and knowledge base

### Key Topics

1. **Getting Started** - Authentication, role selection, navigation
2. **Report Processing** - Upload, ingestion, normalization workflows
3. **Caseload Management** - Assignment, routing, work queue processing
4. **Dashboards & Analytics** - KPIs, performance metrics, leadership views
5. **QA & Compliance** - Quality assurance workflows and compliance tracking
6. **Help Ticket Center** - Issue intake, routing, and resolution
7. **Knowledge Base** - In-app documentation and technical guides
