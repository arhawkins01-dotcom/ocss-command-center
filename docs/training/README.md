## QA & Compliance System Overview

The **Automated Quality Assurance System** (v1.4.0) provides:

### Key Features

- **Automated Case Sampling** - 5 cases per worker per report (deterministic, reproducible)
- **Ohio Compliance Criteria** - OAC 5101:12, ORC 3111.04, OCSE guidance alignment
- **Supervisor QA Review Tab** - Dedicated interface for quality review
- **Executive QA Dashboard** - Agency-wide compliance metrics and trending
- **Real-Time Scoring** - Immediate compliance scoring and feedback

### Roles Supported

- **Supervisors** - QA review interface with compliance checklist
- **Executives** - QA & Compliance dashboard with strategic metrics
- **Support Officers** - Automatic QA sampling on report submission

## Compliance Framework

### Ohio Standards Supported

- **OAC 5101:12** - Administrative procedures for child support services
- **ORC 3111.04** - Paternity establishment requirements
- **OCSE Guidance** - Federal Office of Child Support Enforcement standards

### Compliance Criteria

1. **Timeliness** - Case processing within required timeframes
2. **Documentation** - Complete and accurate case documentation
3. **System Entry** - Proper data entry in SETS and OnBase
4. **Follow-Up Actions** - Appropriate next steps and escalations
5. **Compliance with Procedures** - Adherence to OCSS operational procedures
