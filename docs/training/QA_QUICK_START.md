# QA System - Quick Start (2 Minutes)

## The Problem
> "You are not assigned as a supervisor for any unit. QA review is available to unit supervisors."

## The Solution
You need to select a supervisor that exists for a unit.

---

## 🚀 Get QA Working in 2 Minutes

### Step 1: Process Cases as Support Officer (1 min)

1. **Role:** Change to "Support Officer" 
2. **Unit:** (doesn't matter which one)
3. **Process cases:**
   - Go to "📝 My Assigned Reports" 
   - Mark at least 5 cases as "Completed"
4. **Submit:** Click "✅ Submit Caseload as Complete"
5. **Confirm:** You see "🎯 QA samples automatically generated" ✅

### Step 2: Review QA Samples as Supervisor (1 min)

1. **Role:** Change to "Supervisor"
2. **Select Supervisor:** Look for supervisor name in dropdown
   - Will show names like: `"Establishment Unit 15 Supervisor"`
   - Or: `"Case Maintenance A-E Supervisor"`
3. **New Tab:** You'll see "🎯 QA Review" tab (this is NEW!)
4. **Select Report:** Pick the report you just submitted
5. **See QA Samples:** Will show "5 cases sampled (20%)"
6. **Click a case:** Opens compliance checklist against Ohio rules
7. **Mark completion:** Add notes and save review ✅

### Step 3: View Metrics as Director (optional)

1. **Role:** Change to "Director"
2. **New Tab:** See "🎯 QA & Compliance" tab  
3. **View Metrics:** See agency-wide compliance scores and trends ✅

## 🆘 Troubleshooting

| Problem | Solution |
|---|---|
| Can't find Supervisor dropdown | Make sure "Supervisor" role is selected |
| Supervisor dropdown is empty | Reload page (Ctrl+R or Cmd+R) |
| "No reports with QA samples" error | Go back to Step 1—submit a report first |
| "No reviews completed yet" (in Director tab) | That's fine—you haven't reviewed any yet |
